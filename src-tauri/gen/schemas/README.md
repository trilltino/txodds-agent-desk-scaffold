# src-tauri/gen/schemas

Tauri-generated JSON schemas live here.

## Contents

- `desktop-schema.json`: schema for desktop Tauri configuration.
- `capabilities.json`: schema for capability documents.
- `acl-manifests.json`: schema for access-control manifests.
- `windows-schema.json`: schema for window configuration.

## Rules

- These files are tooling inputs and should stay machine-readable.
- JSON files here cannot carry comments, so this README documents their purpose.
