# src-tauri/icons/android

Android launcher icon outputs generated by Tauri live here.

## Contents

- `mipmap-*`: density-specific PNG launcher icon folders.
- `mipmap-anydpi-v26`: adaptive launcher icon XML.
- `values`: generated Android icon color resources.

## Rules

- Treat density folders as generated outputs.
- Regenerate through Tauri icon tooling instead of editing individual files.
