# src-tauri/icons

Platform icon outputs generated from `branding/icon-1024.png` live here.

## Contents

- Root PNG/ICO/ICNS files for desktop packaging.
- `android/`: Android launcher icon variants generated by Tauri.
- `ios/`: iOS app icon variants generated by Tauri.

## Rules

- Regenerate these with `npx tauri icon branding/icon-1024.png`.
- Prefer editing the source image in `branding/`, not individual generated icon sizes.
- Keep this directory committed because Tauri packaging expects these files.
