# seller-fan-card

The fan-card seller produces mainstream fan-facing explanations from TxLINE events.

## Manifest

- `coral-agent.toml`: seller identity, fan-card service name, and output defaults.

## Runtime Status

Current fan-card output is generated by Rust/browser fallback payload builders. Future LLM-backed behavior should use this manifest for prompt, tone, and tool constraints while Rust keeps policy control.
